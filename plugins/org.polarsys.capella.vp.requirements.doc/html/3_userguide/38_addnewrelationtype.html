Add a new relation type

Relation types are contained in Capella Types Folders.

First you have to disable the corresponding filter :
In Capella Project Explorer menu choose "Customize View...".

Search the filter "Requirement VP Types" and unmatch it.

Capella Types Folders must now be visible, they are located under Architecture Blocks.
You can create your own custom folder (right-click on Architecture Block, "Add Capella Element > Capella Types Folder").

To create a new relation type, right-click on a Capella Types Folder : "Add Capella Element > Relation Type".
Do not forget to set its name.
The new relation type is now available for requirements allocation.
